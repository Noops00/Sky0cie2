# Valentine Page untuk Widya Maharani

Petunjuk singkat:

- Buka file `index.html` di peramban (double-click atau Open with Browser).
- Klik tombol "Buka Pesan" untuk melihat pesan pribadi dan efek hati.

Fitur tambahan:

- **Musik latar:** Klik `Play Musik` dan pilih file audio dari komputer (opsional). Setelah dipilih, musik akan loop.

- **Musik latar:** Klik `Play Musik`. Aplikasi akan otomatis mencoba memuat file `music.mp3` di folder yang sama; jika tidak ada, akan meminta Anda memilih file audio dari komputer. Setelah dipilih atau dimuat, musik akan loop.
- **Tambah foto:** Klik area foto di kiri atas kartu untuk mengunggah foto personal.
- **Konfeti & efek:** Konfeti otomatis muncul saat `Buka Pesan` ditekan.
- **Cetak / Simpan:** Gunakan tombol `Cetak / Simpan` untuk menyimpan sebagai PDF atau mencetak kartu.
- **Bagikan:** Tombol `Bagikan` menggunakan API share bila tersedia atau menyalin tautan.
